About the extension

This extension plays a sound from the vtuber Nakiri Ayame directly in your browser.

How to use

To use "Ayame Noises" use this step-by-step guide:

  1. Install the extension from the Chrome Webstore or Firefox Add-ons
  2. Click on the extension icon in the toolbar
  3. Enjoy the sound!

Privacy Policy

This extension does not collect any data from the user. The extension only plays a sound from the vtuber Nakiri Ayame. The source-code of the project can be found by using the GitHub link above, since the entire project is maintained on GitHub.
